# Container

The container component is used to constrain the contents width while keeping it fluid.

The container component is a wrapper that sets the `max-width` with padding left and right, meaning the content will fill the screen on smaller devices but be centrally fixed when it reaches the `max-width`.

```
import { Container } from "to define"
```

## Usage

```
<Container>
Lorem ipsum dolor sit amet consectetur adipisicing elit. Suscipit, magnam.
</Container>
```
